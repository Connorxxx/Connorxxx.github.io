Hexo+Github 部署教程

Hexo 介绍

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown 解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

本地搭建

安装 Hexo 非常简单, 只需几分钟就能安装完成

安装前提

安装之前请确保已经有下列程序:

  • Node.js
  • Git

Mac 用户可以使用 brew 安装:

安装 Node.js:

brew install node

安装 Git:

brew install git

安装 Hexo

执行下列命令安装:

npm install -g hexo-cli

安装成功后执行下列命令新建本地博客存放文件夹:

hexo init myBlog
cd myBlog
npm install

完成之后运行 hexo s 命令, 在浏览器输入 http://localhost:4000 就可以看到博客了.

部署到 GitHub Pages

点击 New repository 创建一个仓库, 仓库名为 <你的用户名>.github.io

配置 SSH key

获取 SSH Key

cat ~/.ssh/id_rsa.pub

如果之前没有配置, 执行下列命令配置本地 git 账户

git config --global user.name "用户名"
git config --global user.email "邮箱地址"

生成密钥 SSH key

ssh-keygen -t rsa -C '邮箱地址

完成后输入 cat ~/.ssh/id_rsa.pub 获取 SSH Key

获取之后在 Github 上添加 SSH Key, 步骤如下:

Settings -> SSH and GPG keys -> New SSH keys -> Add SSH keys

开始部署

修改 _config.yml 文件:

安装部署插件 hexo-deployer-git

npm install hexo-deployer-git --save

执行下列命令就可以部署上传到 Github 了:

hexo g -d

浏览器访问: 你的用户名.github.io 就能看到你的博客了.

写作

执行下列命令来创建一篇新文章

hexo new 文章名称 或者 hex n 文章名称

文章文件保存在 /source/_posts 中.

执行下列命令就可以在 http://localhost:4000 上看到文章了

hexo g
hexo s

最后, 执行 hexo clean && hexo g -d 命令部署在 Github 上

浏览器打开 你的用户名.github.io 就完成了.
